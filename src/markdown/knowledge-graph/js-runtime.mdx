## 知识体系概览

本体系解决 JavaScript 语言机制、运行时模型与手写实现三类问题，用于支撑面试回答和工程排障。

## 能力边界

- 覆盖范围：类型系统、ES6 语法、作用域闭包、原型链、this 绑定、异步编程、请求模式、内存管理与高频手写题。
- 非覆盖范围：框架专属语法、浏览器渲染细节、网络协议底层实现。

## 核心子主题

1. 数据类型与类型转换：回答值比较、检测与隐式转换的核心机制。
2. ES6 语法与语言特性：回答 let/const、解构、箭头函数、模块化等高频问法。
3. 作用域链与闭包：回答变量可见性、闭包收益与泄漏边界。
4. 原型链与面向对象：回答继承、原型查找与对象创建机制。
5. this、call、apply、bind：回答调用方式与上下文绑定。
6. 事件循环与异步：回答 Promise、async/await、任务队列与执行顺序。
7. 内存管理与请求模式：回答垃圾回收、内存泄漏与 AJAX/fetch 实战。
8. 手写 JS 基础能力：回答面试中的手写实现与场景题。

## 推荐学习路径

- 入门顺序：数据类型与类型转换 -> 作用域链与闭包 -> 原型链与面向对象 -> this 绑定 -> 异步模型。
- 面试优先顺序：作用域闭包 -> 原型链 -> this 绑定 -> Promise 与 async/await -> 手写基础题。
- 项目落地顺序：请求模式治理 -> 异步错误处理 -> 内存泄漏治理 -> 手写工具沉淀。

## 常见混淆边界

- 词法作用域 vs this 绑定：前者在定义时确定，后者在调用时确定。
- Promise 链式调用 vs async/await：前者偏组合表达，后者偏同步风格语义。

## 相关知识岛

- 浏览器原理体系：事件循环与渲染调度直接关联。
- 计算机网络体系：请求模式、重试策略和状态码语义直接关联。
- 前端工程化体系：模块化、构建优化和发布质量门禁直接关联。
