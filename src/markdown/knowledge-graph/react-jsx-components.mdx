## 代码

```tsx
import { forwardRef, useRef } from 'react'

const SearchInput = forwardRef<HTMLInputElement, { onSearch: () => void }>(
  ({ onSearch }, ref) => (
    <>
      <input ref={ref} />
      <button onClick={onSearch}>Search</button>
    </>
  ),
)

export default function Page() {
  const inputRef = useRef<HTMLInputElement>(null)
  return (
    <SearchInput
      ref={inputRef}
      onSearch={() => inputRef.current?.focus()}
    />
  )
}
```

### 示例说明

- 这段代码解决什么问题：组件抽象后仍能把父层 ref 安全透传到子层 DOM。
- 关键语句或参数含义：`forwardRef` 建立 ref 通道，`useRef` 保存 DOM 引用。
- 常见错误写法：封装组件后不透传 ref，导致上层无法聚焦和测量。
